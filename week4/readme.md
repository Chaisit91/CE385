
4.1
สร้างฟังก์ชัน opration เพื่อรับชนิดการคำนวณและตัวเลข 2 ค่า
ภายในใช้เงื่อนไข if...else เพื่อเลือกว่าจะบวก ลบ คูณ หรือหาร
จากนั้นใช้ module.exports เพื่อส่งฟังก์ชันออกไปให้ไฟล์อื่นเรียกใช้ได้
โค้ดที่สองใช้ require เพื่อนำฟังก์ชันจากไฟล์ opration มาใช้งาน
แล้วเรียกฟังก์ชันเพื่อคำนวณแต่ละแบบและแสดงผลผ่าน console.log

4.2
การใช้ Callback โดยส่งฟังก์ชันเข้าไปให้ทำงานหลังดึงข้อมูลเสร็จ แล้วนำข้อมูลมาแสดงผล
โค้ดส่วนที่สองใช้ Promise เพื่อจัดการงาน async ทำให้อ่านโค้ดง่ายและควบคุมสถานะได้ดีขึ้น
resolve() จะถูกเรียกเมื่อดึงข้อมูลสำเร็จ และส่งข้อมูลกลับไปใน .then()
reject() จะถูกเรียกเมื่อเกิดข้อผิดพลาด และส่ง error ไปใน .catch()
ตัวแปร success ใช้จำลองสถานการณ์ว่างานสำเร็จหรือไม่ เพื่อทดสอบการจัดการ error


4.3
สร้างฟังก์ชัน simulateAsyncOperation เพื่อจำลองงานแบบ async โดยถ้าเวลา timeout น้อยกว่า 1000 จะ error แต่ถ้ามากกว่าจะสำเร็จ
ฟังก์ชัน performAsyncTask ใช้ async/await เพื่อรอผลลัพธ์ และใช้ try...catch ดัก error ก่อนแสดงผล โดย 1500 จะสำเร็จ ส่วน 500 จะ error

4.4 
Promise.any() จะคืนค่าจาก Promise ตัวแรกที่ สำเร็จ (resolve) โดยไม่สนตัวที่ล้มเหลว และถ้าทุกตัวล้มเหลวจะส่ง error รวมกลับมา
เหมาะกับงานที่ต้องการผลลัพธ์ที่ใช้ได้เร็วที่สุดจากหลายแหล่ง

Promise.allSettled() จะรอให้ Promise ทุกตัวทำงานเสร็จ ไม่ว่าจะสำเร็จหรือ error
จากนั้นจะส่งผลลัพธ์ของทุกตัวกลับมา พร้อมบอกสถานะว่า fulfilled หรือ rejected
