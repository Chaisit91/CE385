1. การสร้าง REST API
เป็นการสร้างระบบให้สามารถจัดการข้อมูลนักศึกษาผ่าน HTTP Method เช่น ดู เพิ่ม แก้ไข และลบข้อมูล โดยใช้ Express เป็นตัวจัดการเส้นทางการทำงาน

2. การใช้ express.json()
ใช้สำหรับให้เซิร์ฟเวอร์อ่านข้อมูลที่ส่งมาในรูปแบบ JSON ได้ โดยข้อมูลจะถูกเก็บไว้ใน req.body เพื่อนำไปใช้งานต่อ

3. GET /students
ใช้สำหรับดึงข้อมูลนักศึกษาทั้งหมดจากระบบ และส่งข้อมูลกลับไปให้ผู้ใช้งานในรูปแบบ JSON

4. GET /students/:id
ใช้สำหรับดึงข้อมูลนักศึกษาตามรหัส ID ที่ระบุ หากไม่พบข้อมูลจะส่งสถานะข้อผิดพลาดกลับไป

5. POST /students
ใช้สำหรับเพิ่มข้อมูลนักศึกษาใหม่เข้าสู่ระบบ โดยรับข้อมูลจากผู้ใช้และบันทึกลงในรายการนักศึกษา

6. PUT /students/:id
ใช้สำหรับแก้ไขข้อมูลนักศึกษาตาม ID โดยจะอัปเดตข้อมูลเดิมให้เป็นข้อมูลใหม่ที่ส่งเข้ามา

7. DELETE /students/:id
ใช้สำหรับลบข้อมูลนักศึกษาตาม ID โดยจะนำข้อมูลออกจากรายการนักศึกษาในระบบ
